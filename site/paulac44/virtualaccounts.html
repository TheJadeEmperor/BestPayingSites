Existem muitos bancos virtuais na internet, vou apresentar apenas os principais que vocês vão precisar para receber o dinheiro dos sites PTC

PAYPAL é uma empresa – mais que credível – que permite a transferência de dinheiro com um endereço de e-mail. é um serviço internacional que lida com uma espécie de “moeda electrónica”, sendo possível converter esta para dinheiro a qualquer momento.

O servico é gratuito para contas pessoais.

PAYZA é um e-commerce de negócios que permite que os pagamentos sejam feitos por meio da internet entre os usuários que tenham se registrado para o serviço (Payza membros). Para usuários pessoais, é o método de transferência de dinheiro a nível internacional e uma plataforma para a compra de bens e serviços on-line.

O sistema PAGSEGURO é um banco virtual da UOL, usado para compras em lojas virtuais e transferências para bancos do Brasil. A maioria dos ptcs nacionais utiliza o Pagseguro para pagar os seus membros

Pagseguro é um sistema de pagamentos online que permite que indivíduos e empresas possam enviar e receber pagamentos com eficiência e segurança.
